## 🎯 Trilhas de Estudo e Conceitos Aplicados

Este repositório é dividido por módulos e tecnologias. Ao longo dos projetos, aplico e consolido os seguintes conceitos arquiteturais:

### 🚀 Módulo 1: Ecossistema Flutter & Dart
- **UI/UX:** Construção de interfaces complexas utilizando Árvore de Widgets, Slivers e Animações implícitas/explícitas.
- **Gerenciamento de Estado:** Estudos aprofundados em `Provider`, `Bloc` e `Cubit` para aplicações escaláveis.
- **Arquitetura:** Clean Architecture adaptada para Flutter e injeção de dependências (GetIt).
- **Integrações:** Consumo de APIs RESTful, Firebase Authentication e Cloud Firestore.

### ⚛️ Módulo 2: Ecossistema React Native
- **Fundamentos:** Componentização com React, Hooks avançados e navegação nativa com React Navigation.
- **Ecossistema:** Utilização do `Expo` para aceleração do fluxo de desenvolvimento e testes.
- **Gerenciamento de Estado:** Fluxos globais de dados utilizando `Zustand` ou `Context API`.
- **Estilização:** Criação de interfaces consistentes utilizando `NativeWind` (Tailwind) e Styled Components.
